Why AI Workbench

What makes it different?

There are plenty of AI tools out there — but most store your data on their servers.

Common Pain Points 😰

Problems with Web AI Tools

Every word you say to AI is stored on someone else's server. Your work content, private thoughts, and business secrets are beyond your control.

  • Conversations stored on third-party servers
  • Service providers may access your data
  • Chat history disappears when services shut down
  • Price hikes and policy changes — no control
Our Solution 😌

How AI Workbench Does It

All conversations live on your local machine. AI Workbench is just a bridge — it sends your messages to AI providers, shows you the replies, and stores nothing in between.

  • All conversations saved on your local computer
  • API key encrypted and stored locally
  • No intermediate servers, no account system
  • Back up anytime to USB or cloud drive
Get Started in 3 Steps

From Install to First Chat

No technical background needed. Just follow the steps.

1

Install the App

Download the installer for your system — Windows: run the .exe; macOS: open the .dmg and drag to Applications; Linux: launch the .AppImage.

2

Configure an AI Model

Click "Model Config" in the sidebar, pick your AI provider (DeepSeek, ChatGPT, etc.), enter your API Key from the provider's website, and click "Test Connection" to verify.

3

Start Chatting

Click "Chat" in the sidebar, type your message and press Enter. The AI's reply appears word by word. Click "+ New" anytime to start a fresh conversation.

🤔 How to Use Skills?

  1. Auto-Match: Enabled by default. When your message contains trigger keywords (e.g., "write a daily report"), the matching skill activates automatically.
  2. Manual Selection: Pick a skill from the dropdown menu next to the input box to force it for a specific message.
  3. Single-Use: Skills only affect your current message — it switches back to default after sending. No impact on the rest of the conversation.
  4. Create Your Own: Go to the "Skill" page, click "Add Skill," and describe in plain language how you want AI to respond. Your custom skill is ready instantly.

Security & Privacy

Your Data, Your Rules

We put data security first — it's not an afterthought.

🔐

API Key Encrypted Storage

Your API keys are encrypted before being saved locally. The UI shows only a masked format like sk-****xxxx. Not even you can see the full key in the UI — let alone anyone else.

💾

Chat History Is Local Only

All conversations are stored in a local database file (app.db). AI Workbench never uploads your chats to any server. There's no account system — because there's no need for one.

📂

File Access Requires Your Permission

AI cannot automatically read any file on your computer. Only when you actively click "Select File" in the chat and pick a specific file can AI access it. Full file content is never stored — only a brief preview.

⚠️

Five-Layer Command Execution Protection

The "Command Execution" feature is off by default. You must turn it on in settings and specify which folders are allowed. Every command requires confirmation before execution, and dangerous operations (like file deletion) are blocked automatically.

Auto-matching relies on trigger keywords. Check these:

1. Does your message contain the trigger word? (e.g., "Daily Report" needs "daily" or "report" in the message)
2. Is the target Skill enabled? (Check the "Skill" page — is the switch on?)
3. Is the input box selector set to "Auto-Match" mode?

If auto-matching is unreliable, manually pick the skill from the dropdown — manual selection always takes priority.
Default storage locations:
· Windows: %APPDATA%/ai-workbench/app.db
· macOS: ~/Library/Application Support/ai-workbench/app.db

You can see the exact path in the Settings page. We recommend regularly copying this .db file to a USB drive or cloud storage as a backup. You can also change the storage location in Settings to point directly to your sync folder.
File reading has a few limitations:

1. You must use the "Select File" button in the chat — manual path entry won't work.
2. Only .txt, .md, and .markdown formats are supported.
3. Files must be under 5 MB.
4. File authorization is cleared when the app closes — you'll need to re-select files after restarting.
On Windows, the uninstaller will ask whether you want to also delete your local data (chat history, model configs, keys, etc.). You can choose to keep it — reinstall later and your data will still be there.

To be extra safe, manually locate and back up the app.db file before uninstalling, or use "Clear Session History" in Settings before uninstalling.
